# GNOMON EXPERTS - Εφαρμογή Εκθέσεων v2

Web εφαρμογή για συμπλήρωση και παραγωγή εκθέσεων πραγματογνωμοσύνης οχημάτων.

## ΝΕΑ ΧΑΡΑΚΤΗΡΙΣΤΙΚΑ v2

- ✓ Δυναμικές γραμμές ανταλλακτικών (κουμπιά + / −)
- ✓ Δυναμικές γραμμές εργασιών (κουμπιά + / −)
- ✓ Δυναμικός αριθμός ημερομηνιών επίσκεψης
- ✓ Στην εκτύπωση: ένα κελί για "ΑΝΤΙΚΑΤΑΣΤΑΣΗ [όνομα] (τύπος)"
- ✓ 3 έτοιμα templates παρατηρήσεων (επιλογή & επεξεργασία)
- ✓ Φωτογραφικό υλικό με caption, 4 ανά σελίδα
- ✓ Αυτόματη συμπίεση εικόνων για email-friendly μέγεθος

## ΑΡΧΕΙΑ

- **app.py** - Η εφαρμογή Streamlit
- **ekthesi_clean.xlsx** - Το Excel template (μην το αλλάξεις)
- **README.md** - Αυτό το αρχείο

## ΕΓΚΑΤΑΣΤΑΣΗ (μία φορά)

```
pip3 install streamlit openpyxl Pillow
```

**Νέο πακέτο**: Pillow (για επεξεργασία εικόνων)

Για το PDF:
- **Mac**: Κατέβασε LibreOffice από https://www.libreoffice.org/download/
- **Linux Mint**: `sudo apt install libreoffice`

## ΕΚΤΕΛΕΣΗ

```
cd ~/Desktop/interlifesample
streamlit run app.py
```

Θα ανοίξει στον browser: **http://localhost:8501**

## ΧΡΗΣΗ

### 1. Στοιχεία Ατυχήματος & Οχήματος
Συμπληρώστε όπως πριν.

### 2. Επισκέψεις (νέο!)
- Πατήστε **➕ Προσθήκη επίσκεψης** για κάθε νέα επίσκεψη
- Πατήστε **➖ Αφαίρεση επίσκεψης** για να αφαιρέσετε

### 3. Ανταλλακτικά (νέο!)
- Πατήστε **➕ Προσθήκη γραμμής** για να προσθέσετε νέο ανταλλακτικό
- Πατήστε **➖ Αφαίρεση γραμμής** για να αφαιρέσετε
- Γράψτε το όνομα ανταλλακτικού (π.χ. "ΠΡΟΦΥΛΑΚΤΗΡΑΣ ΠΙΣΩ")
- Επιλέξτε τύπο (ΓΝ/ΜΤΧ/ΙΜ)
- Η λέξη "ΑΝΤΙΚΑΤΑΣΤΑΣΗ" προστίθεται αυτόματα στην έκθεση

### 4. Εργασίες (νέο!)
- Ίδιο σύστημα με τα ανταλλακτικά
- Επιλέξτε τύπο (ΕΞΑΓΩΓΗ-ΤΟΠΟΘΕΤΗΣΗ / ΕΠΙΣΚΕΥΗ)
- Γράψτε περιγραφή

### 5. Παρατηρήσεις (νέο!)
- Τσεκάρετε ένα ή περισσότερα από τα 3 templates:
  1. Εμπορική αξία οχήματος
  2. Αρχική εκτίμηση (πριν αποσυναρμολόγηση)
  3. Τελική εκτίμηση (συμφωνημένη)
- Το κείμενο εμφανίζεται στο πεδίο - **μπορείτε να το επεξεργαστείτε**
- Π.χ. στο template 3, αλλάξτε το `___` με το πραγματικό ποσό

### 6. Φωτογραφικό Υλικό (νέο!)
- Σύρετε ή επιλέξτε φωτογραφίες (JPG/PNG)
- Γράψτε **caption** για κάθε φωτογραφία
- Στην έκθεση μπαίνουν **4 φωτογραφίες ανά σελίδα**
- Οι εικόνες συμπιέζονται αυτόματα για email

### 7. Εξαγωγή
- **📊 Δημιουργία Excel**: Επεξεργάσιμο αρχείο
- **📄 Δημιουργία PDF**: Έτοιμο για εκτύπωση/email
- Στο PDF θα δείτε το μέγεθος (στόχος: <10 MB για email)

## ΣΗΜΕΙΩΣΕΙΣ
